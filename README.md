## 为什么需要Clay？

你有没有这样的经历：

- 突然想起某个朋友在某家公司工作，但翻遍通讯录就是找不到
- 会议前5分钟才想起来查对方背景，结果什么都没准备好
- 半年没联系的潜在客户突然换了工作，你毫不知情
- 想给朋友的孩子送礼物，却记不清孩子叫什么名字

这些场景，在职场人、创业者、销售、猎头的日常里太常见了。人际关系是最宝贵的资产，但我们的大脑容量有限，记不住太多细节。传统的通讯录只是一个个名字和电话号码的列表，无法帮你真正"管理关系"。

Clay做的，就是把这些零散的信息串联起来，让你对每个人的了解不止停留在表面。

## Clay到底能做什么？

### 1. 自动聚合，一个人的完整画像

Clay会自动从你的邮件、日历、LinkedIn、Twitter（X）、Facebook、iMessage等平台抓取联系人信息，然后整合成一个完整的档案。

比如说，你在LinkedIn上加了一个行业大佬，Clay会自动把他的工作经历、教育背景、最近发的帖子全都拉过来。你在邮件里跟某个客户聊过项目，Clay也会记录下来。这样，每次打开一个人的资料，你看到的不只是手机号，而是一个立体的、有血有肉的"人"。

### 2. 智能提醒，不错过任何重要时刻

Clay有个很贴心的功能叫"Review"（复盘）。它会定期提醒你：

- 谁的生日快到了
- 哪些人你已经很久没联系
- 谁最近换了工作或搬了家

这些提醒不是生硬的系统通知，而是基于你的社交习惯和关系亲密度智能生成的。你可以在通勤路上花几分钟看看这些提醒，然后发一条问候，维护感情。

职场上最重要的能力之一，就是"记得别人"。Clay帮你做到这一点，而且毫不费力。

### 3. AI助手Nexus，你的人脉导航仪

2023年，Clay推出了AI功能Nexus，这是它的杀手锏。

你可以问Nexus一些神奇的问题：

- "我认识的人里，谁在纽约工作？"
- "谁能帮我介绍OpenAI的人？"
- "这周末去旧金山，应该约谁见面？"

Nexus会从你的整个人脉网络里筛选出答案，甚至还能帮你起草邮件、推荐礼物。它就像一个24小时在线的社交顾问，比你自己还了解你的人脉。

### 4. 随手记笔记，细节决定成败

每次见完人，你可以在Clay里快速记下一些关键信息：

- "他家有两个孩子，大儿子叫Alex"
- "喜欢喝精酿啤酒"
- "对区块链项目感兴趣"

这些看似琐碎的细节，在下次见面时就是打开话题的钥匙。你能记住别人说过的话，别人自然会觉得你用心、靠谱。

**常见问题解答**

**Q: Clay会不会自动发消息给我的联系人？**  
A: 不会。Clay只是帮你整理和提醒，所有沟通都由你自己控制。

**Q: 我的数据能导出吗？**  
A: 可以。你随时可以导出所有数据，不用担心被平台锁定。

**Q: 免费版够用吗？**  
A: 如果你的联系人少于1000个，免费版完全够用。超过的话建议升级Pro。

**Q: 我可以取消订阅吗？**  
A: 随时可以。在设置里点"Cancel Plan"就行，不需要联系客服。

**Q: 支持中文吗？**  
A: 目前界面主要是英文，但可以管理中文联系人信息。
